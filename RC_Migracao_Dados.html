🔄 Migração de Dados — RC Sheet Central

Este ficheiro lê os dados que estão no localStorage deste browser e envia-os para o Google Sheet central.

Abra este ficheiro no mesmo browser/computador onde os questionários foram preenchidos. Execute uma única vez.

Cada registo é enviado uma vez. Pode executar várias vezes sem duplicar — o Sheet central guarda tudo.
